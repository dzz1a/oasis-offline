# Oasis Offline - 离线绿洲心理互助平台

一个面向青少年与大学生群体的情绪支持、社交陪伴与心理成长平台。  
项目聚焦于「低压力表达」「情绪记录」「社区互助」与「心理资源获取」，尝试构建一个更加温和、安全、长期可持续的数字化心理绿洲。

---

# ✨ 项目简介

在高压力学习环境、社交焦虑、休学经历、就业压力与长期孤独感逐渐普遍化的背景下，传统社交平台往往难以提供真正温和且低负担的表达空间。

**Oasis Offline（离线绿洲）** 希望通过：

- 温柔的界面设计
- 低攻击性的社区氛围
- 情绪可视化记录
- 资源整合与陪伴机制
- 更轻量的社交体验

帮助用户建立稳定的心理支持空间。

# 📖 项目理念

> “我们相信，暂停，是为了更好地出发。  
> 所以，我们为歇脚的旅人搭建了这片绿洲。”

Oasis Offline 希望做的不是“解决所有问题”。

而是：

在用户最疲惫、最孤独、最不想开口的时候，  
依然能提供一个温和而安静的空间。
