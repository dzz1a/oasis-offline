# 离线绿洲 - 心理健康平台

一个专为初高中休学、大学生社交和就业的心理健康帮助平台。

## 功能特性

- 🏠 **首页** - 平台介绍、功能特性展示、最新动态、热门资源
- 💬 **绿洲社区** - 论坛发帖、评论互动、热门标签
- 💭 **聊天** - 私信聊天、联系人列表、在线状态
- 📚 **资源中心** - 心理文章、音频资源、视频课程
- 🎯 **情景模拟** - 面试/社交/压力场景模拟
- 👤 **个人中心** - 用户资料、能量状态、情绪记录、紧急联系人

## 技术栈

- React + TypeScript
- Styled Components
- React Hooks

## 开始使用

### 安装依赖

```bash
npm install --legacy-peer-deps
```

### 启动开发服务器

```bash
npm start
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
src/
├── components/         # 组件
│   ├── layout/       # 布局组件
│   └── ui/           # UI组件
├── pages/            # 页面组件
├── data/             # Mock数据
├── styles/           # 主题样式
└── types/            # TypeScript类型定义
```

## 协作指南

### 获取项目

1. 克隆仓库：

```bash
git clone <仓库地址>
```

2. 安装依赖：

```bash
npm install --legacy-peer-deps
```

### 开发流程

1. 创建功能分支：

```bash
git checkout -b feature/你的功能名称
```

2. 提交代码后推送到远程：

```bash
git add .
git commit -m "描述你的更改"
git push origin feature/你的功能名称
```

3. 创建 Pull Request
