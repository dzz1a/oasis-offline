# 「离线绿洲」项目交接文档

## 📋 项目概述

### 项目名称
离线绿洲 - 心理健康支持与社区交流平台

### 项目目标
为休学中学生、有社交/就业问题的大学生提供心理健康支持和社区交流平台

### 核心功能
- 用户注册/登录
- 论坛帖子发布与互动
- 个人资料编辑
- 好友添加与管理
- 实时聊天
- 关注功能
- 资源中心
- 情景模拟

## 🚀 本地运行

### 环境要求
- Node.js >= 20.x
- npm >= 10.x

### 安装依赖

**前端依赖**:
```bash
npm install
```

**后端依赖**:
```bash
cd backend
npm install
```

### 启动服务

**启动前端**:
```bash
npm start
```
访问: http://localhost:3000

**启动后端**:
```bash
cd backend
npm start
```
服务端口: http://localhost:5000

## 🔐 认证机制

### JWT Token
- Token存储位置: localStorage
- Token有效期: 7天
- Token名称: `token`

### 用户认证流程
1. 用户登录 → 获取JWT Token
2. 请求API时在Header中携带Token
3. 后端验证Token有效性

## 📡 API接口

### 用户相关
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/users/register | 注册 |

## 🔄 关注与好友关系

### 关系逻辑
1. **好友关系 = 互相关注**
   - 接受好友请求时自动互相关注
   - 删除好友时自动取消互相关注

2. **单向关注**
   - 通过绿洲社区可关注非好友用户
   - 关注者数量统计包含好友和单向关注者

### 统计数据
- **关注者数**: 好友数 + 通过绿洲关注你的人数
- **已关注数**: 好友数 + 你通过绿洲关注的人数
